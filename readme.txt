1. Получить token на сайте https://vk.com/dev и вписать его номер в файл с названием 'access-token vk'.
2. Вписать id аккаунта, с которого будут скачаны фото, в файл с названием 'user_id'.
3. Запустить первую часть кода с названием 'first_part.py' и убедиться, что файлы (фото) скачаны на жесткий диск в пап-
ку с названием 'images', также убедиться в наличие файла с названием 'list_photos.json'.
4. Получить token на сайте https://yandex.ru/dev и вписать его номер в файл с названием 'access-token yandex'.
5. Запустить вторую часть кода с названием 'second_part.py' и убедиться в том, что файлы (фото) загрузились на я.диск.

P.S.: Все номера token, id вписывать в том виде, в котором их предоставляет конкретный ресурс, в файлы с расширением
.txt и соответствующими названиями.
